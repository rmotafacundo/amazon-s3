# amazon-s3
Exemplo Simples de como consumir a API do Serviço Amazon S3

Criar Bucket;
Listar Buckets;
Excluir Buckets;
Listar arquivos de um Bucket;
Download de arquivos(PDF,XML e imagem);
Upload de Arquivos;
Excluir arquivos;

Exemplo feito de forma simplista, sem considerar boas práticas de programação. Apenas para servir de fonte de estudos,
pra quem se interessar.
